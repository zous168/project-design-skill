# CRM 非功能性需求(NFR)

> 全闭环产出 · 对应 skill 文档 **08**。上游:[01 需求规格](01-需求规格.md);影响 [03 架构](03-系统架构.md)、[09 技术选型](09-技术选型.md)。每项**可度量、可验证**。

| 维度 | 指标(可验证) | 验证方式 |
|------|----------------|----------|
| **性能** | 线索查重 / 话术检索 **P95 ≤ 200ms**;工作台列表 P95 ≤ 500ms | 压测 1k RPS |
| **容量** | 线索 ~50w/年、商机 ~10w/年、跟进 ~200w/年;logs ~千万/年 | PG 容量模型 |
| **并发** | 销售高峰 ≥ 200 坐席在线、≥ 100 RPS 写 | 压测 |
| **可用性** | 核心查询接口 ≥ **99.9%**;Hermes 侧可降级为"仅记录留资" | SLO 监控 + 降级演练 |
| **异步 SLA** | 赢单回写 ≤ 5s 完成(P95);DLQ 非空告警 ≤ 1min | 队列指标(见 [07](07-异步处理.md)) |
| **安全** | API Key/JWT 鉴权;联系方式**脱敏展示**;全量操作审计→ logs | 安全测试 + 审计核查 |
| **隐私** | PII 字段(手机号/customerId 关联)分级、访问留痕;私有化数据不出域 | 字段分级核查(见 [05](05-数据设计.md)) |
| **可观测** | 接口/队列/调度 指标 + 告警 + traceId 贯穿 | 监控看板 |
| **可维护/运维** | 模块化单体边界清晰;配置中心 cfg;蓝绿/滚动发布 | 代码评审 + 发布演练 |
| **可用性(UX)/可达性** | 键盘可达、对比度 WCAG AA、reduce-motion | UI pre-flight(Rubric D) |
| **响应式** | 三档断点:桌面 ≥1025 / 平板 ≤1024 / 移动 ≤640,各端不破版 | 多视口实测(见 [响应式设计](响应式设计.md)) |
| **国际化** | **默认中英双语**,底层 i18n(vue-i18n + EP locale),**不硬编码**,可扩展更多语言 | 切换 zh/en 验证 |
| **主题** | **亮 / 暗 / 随系统** 三模式,暗色可读 | 三模式实测 |
| **部署可移植** | 本地私有化 与 服务器集中部署同构,能力与位置无关 | 双形态部署 |
| **成本** | 单 PG + Redis 起步,按量扩 schema/实例 | 容量×单价模型 |

- **逻辑:** 性能/容量与 [05 数据设计](05-数据设计.md) 卷量、[03 架构](03-系统架构.md) 缓存/读写策略一致;异步 SLA 对齐 [07](07-异步处理.md)。

# CRM 功能树（模块 → 页面 → 操作）+ 系统角色

> 试跑产出（对应 skill 文档 **04 — Feature Tree**）。功能树是**评审主清单**：每个叶子操作都挂【角色 / FR / 接口 / 原型页面】，供逐行核对覆盖。
> 子系统（Phase 0 确认）：**坐席工作台(Web) · 管理后台(Admin) · 开放API(Hermes)**。

## 系统角色（对应 §需求 §1.2 面向用户；skill 要求 01 角色矩阵）

| 角色 | 代号 | 说明 | 主要子系统 |
|------|------|------|------------|
| 销售坐席 | R1 | 跟进自己名下线索/商机，登记跟进，申请特价 | 坐席工作台 |
| 销售主管 | R2 | 团队视图、审批特价、改派、关闭；含 R1 全部（团队范围） | 坐席工作台 |
| 运营管理员 | R3 | 配置 SOP/话术/角色/渠道分配规则/报表，不直接销售 | 管理后台 |
| 集成账号(Hermes) | R4 | 机器账号：建线索、回写跟进、查 SOP/话术、赢单回调 | 开放 API |

## A. 坐席工作台（Web） — R1 / R2

- **A1 工作台首页** （FR §4.6 / §4.7 · 页面 `index.html`）
  - A1.1 查看待办队列（线索/商机，按优先级/等待/金额排序）— R1/R2 · `GET /workbench/queue`
  - A1.2 推进商机阶段 — R1 · `PATCH /opportunities/{id}/stage`
  - A1.3 改派负责人 — **R2** · `POST /opportunities/{id}/reassign`
  - A1.4 申请/审批特价 — 申请 R1 / 审批 **R2** · `POST /opportunities/{id}/special-price`
  - A1.5 关闭线索/商机 — **R2** · `POST /opportunities/{id}/close`
- **A2 线索管理** （FR §4.1 · 页面 `leads.html`）
  - A2.1 线索列表 + 筛选 — R1/R2 · `GET /leads`
  - A2.2 录入线索（手动）— R1 · `POST /leads`
  - A2.3 批量导入 — R1/R2 · `POST /leads/import`
  - A2.4 分配/改派线索 — R1(认领)/R2(指派) · `POST /leads/{id}/assign`
  - A2.5 线索详情 → 转商机 — R1 · `POST /leads/{id}/convert`
- **A3 商机管理** （FR §4.2/§4.3 · 页面 `opportunities.html`）
  - A3.1 阶段看板 — R1/R2 · `GET /opportunities?view=board`
  - A3.2 新建商机 — R1 · `POST /opportunities`
  - A3.3 登记跟进 — R1 · `POST /opportunities/{id}/follow-ups`
  - A3.4 查看跟进时间线（含 Hermes 触达）— R1/R2 · `GET /opportunities/{id}/follow-ups`
- **A4 客户全景（只读聚合）** （FR §4.6 · 工作台内嵌）
  - A4.1 查看等级/标签/历史/CSM摘要 — R1/R2 · `GET /customers/{id}/panel`（代理客户全景域）

## B. 管理后台（Admin） — R3

- **B1 SOP 规则** （FR §4.5 · 页面 `sop.html`）
  - B1.1 场景模板列表 — R3 · `GET /admin/sop`
  - B1.2 编辑节点规则/阈值 — R3 · `PUT /admin/sop/{code}`
  - B1.3 发布版本 / 灰度 — R3 · `POST /admin/sop/{code}/publish`
  - B1.4 版本历史 — R3 · `GET /admin/sop/{code}/versions`
- **B2 话术与政策** （FR §4.4 · 页面 `playbook.html`）
  - B2.1 话术列表/筛选 — R3 · `GET /admin/playbooks`
  - B2.2 新建/编辑并升版 — R3 · `POST|PUT /admin/playbooks`
  - B2.3 失效 / 上下架（控制 RAG 索引）— R3 · `POST /admin/playbooks/{id}/retire`
- **B3 角色与权限** （FR §4 角色 · 页面 `admin-roles.html`*）
  - B3.1 角色列表 / 权限分配 — R3 · `GET|PUT /admin/roles`
- **B4 渠道与分配规则** （FR §4.1 自动分配 · 页面 `admin-rules.html`*）
  - B4.1 渠道来源配置 — R3 · `GET|PUT /admin/channels`
  - B4.2 分配规则（区域/负载/技能组）— R3 · `PUT /admin/assign-rules`
- **B5 运营报表配置** （FR §4.7 · 页面 `analytics.html`）
  - B5.1 漏斗/绩效/话术分析看板 — R3 · `GET /admin/analytics`

## C. 开放 API（Hermes 集成） — R4（无界面，出接口文档）

- **C1 线索接口** — C1.1 建线索(幂等) `POST /api/v1/leads` · C1.2 查重 `GET /api/v1/leads:dedup`
- **C2 跟进回写** — C2.1 写跟进(带 traceId) `POST /api/v1/follow-ups`
- **C3 SOP 查询** — C3.1 按场景编码取节点/权限/话术 `GET /api/v1/sop?code=`
- **C4 话术检索** — C4.1 RAG 检索销售素材 `POST /api/v1/playbooks:search`
- **C5 商机回调** — C5.1 赢单事件回调 `POST /api/v1/opportunities/{id}:won`（触发客户全景/CSM）

> 标 `*` 的为本次未做原型的后台页面（已列入屏清单，留待管理后台子系统原型）。

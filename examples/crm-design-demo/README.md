# 全渠道销售 CRM — 分析与设计

全渠道销售体系的**销售域业务底座与规则管控中心**:统一留资/线索入口、标准化商机与跟进、沉淀话术与 SOP、管控报价折扣,并为 AI 智能层 Hermes 提供结构化数据与规则边界。面向销售坐席 / 主管 / 运营,三子系统(坐席工作台 Web · 管理后台 Admin · 开放 API)。技术栈:**Vue 3 + Element Plus / NestJS / PostgreSQL + Redis(BullMQ),模块化单体**。

> **Set version: v1.1**　·　由 `project-design-docs` skill 全闭环产出(Phase 0→6)。
